Possiamo integrare Filestash per lavorare con il tuo backend, i tuoi processi, i tuoi utenti e il tuo marchio Chatta con noi

Esplora e gestisci il tuo server FTP

Il nostro client FTP web rende facile gestire ed esplorare i dati sul tuo server. Funziona con qualsiasi server FTP compatibile come Proftpd, Pureftpd, Filezilla, Vsftpd, ...

Editor e visualizzatore di contenuti

Guarda i tuoi video o immagini, l'editor FTP integrato ti permette di visualizzare e modificare non solo i documenti d'ufficio ma anche una vasta gamma di formati di file e altro ancora

Software libero e Open Source

Filestash è un software open source. Il nostro codice è disponibile su Github in modo da poter controllare, espandere e ospitare la propria versione (vedi la documentazione).

Mac, Windows e Linux

Il nostro strumento software è disponibile come applicazione web dal tuo browser dandoti la massima portabilità senza dover installare niente. Funziona meglio con Chrome, Chromium e Firefox.

Condivisione di file e ricerca full-text

Lavorate con link condivisi che rimangono sotto stretto controllo. Il motore di ricerca full-text rende facile trovare i vostri contenuti esistenti.

Esponi il tuo FTP sul web

Dai ai tuoi utenti l'interfaccia che desiderano e integrala attraverso un iframe o con una pagina personalizzata come questa pagina

Online e offline

Come PWA, Filestash non solo funziona alla grande online, ma si sforza anche di fornire alcune funzionalità di base per l'uso offline

Si integra con il tuo file manager

I link condivisi si integrano con il vostro file manager. Possono essere utilizzati come un'unità di rete sotto Windows, OSX o Linux. Questi collegamenti sono un server WebDAV completo che si collega al tuo FTP

Estensibile

Il nostro strumento web FTP può essere interamente personalizzato. Può essere etichettato ed esteso attraverso il meccanismo dei plugin per adattarsi ai vostri bisogni e processi con molta flessibilità

Creare uno straordinario client FTP web

I client FTP tradizionali come Filezilla FTP o WinSCP sono ottimi per sysadmin e ingegneri, ma tutti i tuoi utenti potrebbero non avere familiarità con il protocollo sottostante e volere semplicemente uno strumento per fare le cose in modo semplice. La proposta di valore di Filestash è di rendere FTP facile da usare per i vostri utenti finali, apportando caratteristiche di lavoro che rendono l'FTP un sostituto moderno di Dropbox.
