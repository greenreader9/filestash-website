Podemos integrar Filestash para que funcione con tu backend, tus procesos, tus usuarios y tu marca Chatea con nosotros

Explora y gestiona tu servidor FTP

Nuestro cliente FTP web facilita la gestión y exploración de los datos de tu servidor. Funciona con cualquier servidor compatible con FTP, como Proftpd, Pureftpd, Filezilla, Vsftpd, ...

Editor y visor de contenidos

Mira tus vídeos o imágenes, el editor FTP integrado te permite ver y editar no sólo documentos de oficina, sino también una amplia gama de formatos de archivo y más

Software libre y open source

Filestash es un software open source. Nuestro código está disponible en Github para que puedas auditar, ampliar y autoalojar tu propia versión (consulta la documentación).

Mac, Windows y Linux

Nuestra herramienta de software está disponible como aplicación web desde tu navegador, lo que te proporciona la máxima portabilidad sin tener que instalar nada. Funciona mejor con Chrome, Chromium y Firefox.

Compartir archivos y búsqueda de texto completo

Trabaja con enlaces compartidos que permanecen bajo estricto control. El motor de búsqueda de texto completo facilita la búsqueda de los contenidos existentes.

Destaca tu FTP en la web

Consigue para tus usuarios la interfaz que se merecen e intégrala a través de un iframe o con una página personalizada como ésta

Online y offline

Como PWA, Filestash no sólo funciona muy bien en línea, sino que también se esfuerza por ofrecer algunas funcionalidades básicas para el uso sin conexión

Se integra con tu gestor de archivos

Los enlaces compartidos se integran con tu gestor de archivos. Pueden utilizarse como una unidad de red en Windows, OSX o Linux. Estos enlaces son servidores WebDAV completos que se conectan a tu FTP

Extensible

Nuestra herramienta de FTP web es totalmente personalizable. Se puede etiquetar y ampliar a través del mecanismo de plugins para que se adapte a tus necesidades y procesos con mucha flexibilidad

Creación de un impresionante client FTP web

Los client FTP tradicionales, como Filezilla FTP o WinSCP, son estupendos para los administradores de sistemas y los ingenieros, pero es posible que todos tus usuarios no estén familiarizados con el protocolo correspondiente y sólo quieran una herramienta para hacer las cosas de forma sencilla. La propuesta de calidad de Filestash es hacer que el FTP sea fácil de usar para tus usuarios finales, aportando funciones de colaboración para hacer del FTP un sustituto moderno de Dropbox.
